# Projeto de Análise de Dados - Banco Digital (SQL)

Este projeto simula a análise de dados de uma fintech.

## Objetivo
Analisar comportamento financeiro de clientes utilizando SQL.

## Dataset
- clientes.csv
- contas.csv
- transacoes_100k.csv (100 mil registros)

## Perguntas respondidas
1. Qual o saldo total por tipo de conta?
2. Quais contas movimentam mais dinheiro?
3. Como está a movimentação financeira por mês?
4. Quem são os clientes com maior saldo?
5. Qual o ticket médio por tipo de transação?

## Ferramentas
- SQL
- Banco de dados relacional
- Power BI / Tableau para dashboards
